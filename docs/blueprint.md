# **App Name**: Mshkur AI Studio

## Core Features:

- Digital Card Creation: Create a professional digital business card with customizable fields, powered by Google Gemini 2.0. It will function as a tool and allow user customization, image enhancement and vector logo design.
- Data Extraction via OCR: Extract data from images using Google Cloud Vision API (OCR) to pre-fill card information.
- 3D Background Generation: Generate dynamic 3D video backgrounds using React Three Fiber, configurable by the user.
- Logo Generation and Enhancement: Generate abstract logos and enhance their quality to 8K using AI tools.
- Google Drive Integration: Automatically save digital cards and associated resources to the user's Google Drive.
- Viral Referral System: Implement a referral system to track and reward user sharing using Firestore to track UTM parameters.

## Style Guidelines:

- Primary color: Deep, saturated blue (#29ABE2) to convey a sense of trust, reliability, and technical expertise; blue relates to engineering and software.
- Background color: Dark gray (#2B3035), in line with a dark color scheme to enhance the vibrancy of UI elements. Low saturation, professional.
- Accent color: Violet (#9C27B0), approximately 30 degrees away from blue on the color wheel; violet symbolizes creativity and innovation.
- Body and headline font: 'Inter', a grotesque-style sans-serif, for a modern, machined, objective, and neutral look.
- Code font: 'Source Code Pro' for displaying code snippets.
- Use modern, minimalist icons to represent different features and actions.
- Implement smooth, subtle transitions and animations to enhance user experience.
